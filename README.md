# Hospital Management System

## Project Overview

This Hospital Management System is a full-stack web application developed as a university/project learning experience to understand how real hospital systems work. The project manages important hospital operations such as patient registration, doctor management, appointment booking, and billing.

The main purpose of this project was to improve my practical skills in:

* Frontend development
* Backend development
* Database management
* API integration
* CRUD operations
* SQL queries and joins
* Full-stack project structure

This project helped me understand how frontend, backend, and database communicate together in a real-world system.

# Features

## Patient Management

* Register new patients
* Store patient information
* View patient records
* Search patients dynamically

## Doctor Management

* View doctor list
* Department-wise doctor display
* Dynamic doctor loading from database

## Appointment System

* Book appointments
* Select patient and doctor
* Store appointment date and time
* View all appointments

## Billing System

* Generate patient bills
* View billing history
* Dynamic patient dropdown
